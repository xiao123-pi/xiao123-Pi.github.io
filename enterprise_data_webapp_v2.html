⚙️ API 设置

📌 如何获取 PAT 和 Bot ID？ 1. 登录 coze.cn，点击右上角头像 →「API Access Token」→ 创建并复制 Token
2. 进入你的 Bot 编排页面，Bot ID 在页面 URL 中：coze.cn/space/xxx/bot/Bot_ID
3. Bot 需要支持企业搜索能力（如搜索插件）
4. 所有设置仅保存在本地浏览器，不会上传

📖 使用说明

1

配置 API

点击右上角"设置"按钮，填入你的 Coze PAT 和 Bot ID，保存设置。Bot 需要具备企业搜索能力。

2

填写采集参数

在左侧表单中输入目标地区（如"广州白云区"），选择企业规模，设置采集数量（1-100）。

3

开始采集 / 关键词搜索

方式一：填写地区+规模+数量，点击"开始采集"按地区批量采集。

方式二：在"关键词搜索"卡片中输入关键词（如"科技""建筑""餐饮"），可选填地区和数量，点击"关键词搜索"按行业/关键词精准查找。

4

查看与管理数据

采集完成后，数据自动填充到表格中。你可以搜索筛选、删除行、手动添加行，或导出为 JSON / CSV 格式。

📋 12 个采集字段说明

  • 企业名称 — 企业全称
  • 企业规模 — 大型 / 中型 / 小型
  • 法定代表人 — 法人姓名
  • 注册资本 — 如"5000万人民币"
  • 成立日期 — YYYY-MM-DD 格式
  • 所属行业 — 国民经济行业分类
  • 企业类型 — 如"有限责任公司"
  • 员工人数 — 如"100-499人"
  • 经营状态 — 存续 / 开业
  • 数据来源 — 企查查或天眼查详情页URL
  • 联系电话 — 企业联系电话
  • 企业地址 — 企业注册地址
